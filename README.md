# Sunset Bay Discord Server Portfolio

A clean, modern portfolio website showcasing the Sunset Bay Discord community with a sunset-ocean aesthetic.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Sunset Theme**: Beautiful gradient animations and coastal color scheme
- **Modern UI**: Clean typography, smooth animations, and interactive elements
- **Fast Loading**: Optimized CSS with minimal dependencies
- **SEO Friendly**: Semantic HTML structure with proper meta tags

## Structure

- **Hero Section**: Eye-catching introduction with animated sunset gradient
- **About Section**: Community description with coastal styling
- **Features Section**: Four key features with emoji icons and hover effects
- **Stats Section**: Community highlights with gradient background
- **Call-to-Action**: Final invitation to join the Discord server

## Technologies Used

- HTML5 Semantic Structure
- CSS3 with Modern Features (Grid, Flexbox, Animations)
- Google Fonts (Inter)
- No JavaScript Required (pure CSS animations)

## Customization

### Animations
- Floating feature icons
- Gradient background animations

## Usage

Simply open `index.html` in a web browser. No build process or server required.
